# Local installation Guide
 1. Clone repository 
 ```git clone https://github.com/edmundlth/AMSIWS2021_neural_network_workshop```
 2. Install required packages
 ```
 cd AMSIWS2021_neural_network_workshop
 pipenv install . 
 ```
 3. Run `jupyter`
 ```
 pipenv run jupyter notebook
 ```
